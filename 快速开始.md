# RSA加解密系统 - 快速开始

## ?? 3分钟快速上手

### 第一步：启动程序
双击运行 `RSA.exe`

### 第二步：生成密钥（首次使用）
```
首次使用需要生成RSA密钥对。
是否现在生成？(y/n): y

请选择安全级别 (0-4): 2
```
**推荐选择级别2**（中级安全，速度较快）

### 第三步：加密消息
```
请选择操作 (0-5): 2

请输入要加密的文本：
这是一条秘密消息！

密文（请妥善保管）：
228 191 153 230 152 ...（一串数字）
```
**复制这串数字，这就是加密后的密文**

### 第四步：解密消息
```
请选择操作 (0-5): 3

请输入要解密的密文：
228 191 153 230 152 ...（粘贴刚才的数字）

解密后的文本：这是一条秘密消息！
```

## ?? 重要提示

### ? 必须记住
1. **私钥文件** (`private_key.txt`) 千万不要丢失
2. **私钥文件** 千万不要发送给别人
3. **密文** 是一串数字，用空格分隔

### ?? 与他人通信
**发送加密消息：**
1. 获取对方的 `public_key.txt`（公钥）
2. 加载对方的公钥
3. 使用对方的公钥加密消息
4. 发送密文给对方

**接收加密消息：**
1. 使用自己的私钥解密
2. 只有拥有私钥的人才能解密

### ?? 密钥文件位置
程序运行后会在程序目录生成：
- `public_key.txt` - 公钥（可以分享）
- `private_key.txt` - 私钥（严格保密）

### ?? 备份建议
定期备份这两个密钥文件到安全位置！

## ?? 常见问题

**Q: 密钥文件丢失了怎么办？**
A: 需要重新生成新密钥，但旧密钥加密的消息将无法解密。

**Q: 如何给朋友发送加密消息？**
A: 
1. 让朋友生成密钥并发送 `public_key.txt` 给你
2. 你加载他的公钥后加密消息
3. 发送密文给他
4. 他用自己的私钥解密

**Q: 忘记选择了哪个安全级别？**
A: 选择菜单的"4. 查看密钥信息"可以看到模数n的大小。

**Q: 可以加密中文吗？**
A: 完全支持中文、英文、数字和符号。

**Q: 加密很慢怎么办？**
A: 降低安全级别或减少文本长度。日常使用级别1-2即可。

## ?? 安全级别推荐

| 用途 | 推荐级别 | 说明 |
|-----|---------|------|
| 学习测试 | 0-1 | 快速，但不够安全 |
| 日常笔记 | 1-2 | 平衡速度和安全 |
| 重要文件 | 2-3 | 较高安全性 |
| 机密信息 | 3-4 | 最高安全性 |
